# Identity providers

Mostly AI supports a variety of identity providers to allow you to use your existing identity system to authenticate users to MOSTLY AI.

## Supported identity providers

Use the following external documentaiton to configure single sign-on for your organization:

| Provider | Documentation |
|----------|------|
| Email & password | N/A |
| Github | [About authentication with single sign-on](https://docs.github.com/en/enterprise-cloud@latest/authentication/authenticating-with-single-sign-on/about-authentication-with-single-sign-on) |
| Google | [About SSO](https://support.google.com/a/answer/60224?hl=en) |
| Microsoft | [Microsoft Entra single sign-on](https://www.microsoft.com/en-us/security/business/identity-access/microsoft-entra-single-sign-on) |

Individual users can provide their own credentials to access MOSTLY AI using any of the above identity providers without additional setup.

# Quick start: Synthetic data

Synthetic data is artificially generated data that mimics the structure and statistical properties of real data, but contains no actual personal or sensitive information.

Use a [generator](/generators) to create sythetic data based on your requirements. Follow the [generator quick start guide](/quick-start/model-creators) if you want to create your own generator.

<Steps>
## Step 1: Generate synthetic data

<Tabs items={['Manual', 'Assistant', 'Synthetic Data SDK']}>
<Tabs.Tab>
1. Navigate to the generator that you wish to use by clicking **Generators** in the left-side navigation menu and selecting from the available generators.
2. On the generator page, click **Generate data**.
3. Configure the generation parameters based on your requirements:

| Parameter | Description |
|-----------|-------------|
| Sample size | Define the number of rows to generate for your synthetic data. |
| Conditional simulation | Use a seed dataset as context for conditional simulation. |
| Sampling controls | Adjust how closely your synthetic data mirrors the original distribution. |
| Fairness | Apply statistical parity to a target column with respect to selected sensitive columns, ensuring statistical independence. |
| Rebalancing | Rebalance the distribution in a categorical column based on specified categories and their new percentages. |
| Imputation | Replace missing values in a column with statistically coherent values from the same column. |
| Data report | Control data report generation for this model. Disable it to speed up the process of finalizing synthetic dataset generation. |

4. Click **Start generation**.

Follow progress in the **Generation status** section on the synthetic dataset page.

**What's next**

When the generation of synthetic data completes, you can [download](/quick-start/data-consumers#step-2-download-synthetic-data) the data in CSV, Parquet, or XSLX format.

You can also [make the synthetic dataset public](/quick-start/data-consumers#step-3-make-the-synthetic-dataset-public) so others can access its page, review the **Data report**, **Data samples**, **Generation steps**, and **Configuration**, or download the data in their preferred format.

</Tabs.Tab>
<Tabs.Tab>

1. Start a new chat with the Assistant by clicking **New chat** in the left-side navigation menu.
2. Prompt the Assistant to connect to the synthetic data generator that you would like to use. Alternatively, you can click **Explore** from the generator page of any generator.

```bash
Connect to the Baseball Generator and describe this resource.
```

3. Define the synthetic dataset that you would like to generate with this generator.

```bash
Generate a synthetic dataset containing 1000 players with a batting average over 0.300 using this generator.
```

<Callout>

Describe your use case and the resources that you'd like to use and ask the Assistant to provide the code to use with the Synthetic Data SDK.

</Callout>

</Tabs.Tab>

## Step 2: Download synthetic data

<Tabs items={['UI', 'Synthetic Data SDK']}>
<Tabs.Tab>
1. With the synthetic dataset open, click **Download synthetic dataset**.
2. Select to download as CSV, Parquet, or Excel.
    <Image
        src="/docimages/quickstart/data-consumers/download-sd-01-click-download-synthetic-data-select-format.webp"
        alt="MOSTLY AI - Download a synthetic dataset - Click Download synthetic data and select CSV Parquet or Excel"
        width={800}
        height={150}
    />
</Tabs.Tab>
<Tabs.Tab>

<CustomCallout type="API">Available in both [Local and Client modes](/python-sdk#local-and-client-modes).</CustomCallout>

Use the Synthetic Data SDK to easily download your synthetic data in the file format that you need.

```python copy filename="python"
# download as CSV format
sd.download(format="CSV")

# download as Parquet format
sd.download(format="PARQUET")

# download as Excel format
sd.download(format="XLSX")
```

</Tabs.Tab>
</Tabs>

## Step 3: Make the synthetic dataset public

1. With the synthetic dataset open, click **Share**.
   <Image src="/docimages/quickstart/data-consumers/datasets-make-public-01-select-share.webp" alt="MOSTLY AI - Make the synthetic dataset public - 01 - Click Make public" width={800} height={150} />
2. Select **Public**.
   <Image src="/docimages/quickstart/data-consumers/datasets-make-public-02-select-public.webp" alt="MOSTLY AI - Make the synthetic dataset public - 02 - Select Public" width={800} height={150} />
3. Click **Save**.
4. Copy and share the URL of the synthetic dataset.

**What's next**

Use the synthetic dataset to freely analyze the data without the privacy-related concerns.

Share the dataset URL so others can do the same.

# AWS Marketplace Deployment

MOSTLY AI is available via the AWS Marketplace.

## Prerequisites

- An [AWS account](https://portal.aws.amazon.com/billing/signup)

## Deployment

Deployment Overview, Requirements and Prerequisites are available in the [GitHub repository](https://github.com/mostly-ai/aws-marketplace).

You will also find installation [examples](https://github.com/mostly-ai/aws-marketplace/tree/main/examples), which may be used as a step-by-step installation guide via Terragrunt.

## Post-deployment

1. Once the installation has been completed, you can access the Platform via the DNS name (FQDN) used during the setup.

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-11-login-screen.webp" alt="Platform access" width={800} height={400} />

2. Log in using the superadmin credentials:
   - If following an [examples](https://github.com/mostly-ai/aws-marketplace/tree/main/examples)-based installation, the superadmin credentials will be the ones defined in `helm-stack/mostly-combined/terragrunt.hcl`.
   - Otherwise, use the default ones.

**Result**

Now you have a fully functional deployment of MOSTLY AI in your own AWS environment! ðŸŽ‰

## Change Keycloak admin password

1. Access the Keycloak admin console by navigating to `https://<your-load-balancer-dns>/auth/admin`
2. Log in with the default Keycloak admin credentials:

   - Username: `admin`
   - Password: `admin`

3. Navigate to **Manage** > **Users**

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-12-keycloak-users.webp" alt="Keycloak users list" width={800} height={400} />

4. Select the **admin** user

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-13-keycloak-credentials.webp" alt="Keycloak credentials tab" width={800} height={400} />

5. Click the **Reset password** button under **Credentials** tab

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-14-keycloak-reset-password.webp" alt="Keycloak reset password" width={800} height={400} />

6. Set a new secure password for the Keycloak admin account

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-15-keycloak-new-password.webp" alt="Keycloak new password" width={800} height={400} />

## Change superadmin password

1. Select **mostly-generate** from the dropdown menu at the top left

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-16-mostly-generate.webp" alt="Select mostly-generate realm" width={800} height={400} />

2. Navigate to **Manage** > **Users**

3. Select the **superadmin** user

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-17-superadmin.webp" alt="Superadmin users list" width={800} height={400} />

4. Click the **Reset password** button under **Credentials** tab

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-19-superadmin-reset-password.webp" alt="Superadmin reset password" width={800} height={400} />

5. Set a new secure password for the superadmin account

<Image src="/docimages/install/aws-marketplace/01-aws-marketplace-18-superadmin-new-password.webp" alt="Superadmin credentials tab" width={800} height={400} />

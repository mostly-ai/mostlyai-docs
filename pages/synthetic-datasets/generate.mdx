# Generate a synthetic dataset

After you have a [generator](/generators) trained on a single- or multi-table original dataset, you can use it to create a new synthetic dataset.

<Tabs items={['UI', 'Local Python environment', 'Remote platform']}>
<Tabs.Tab>
**Steps**

1. Start a new synthetic dataset in one of two ways.
    * Open a trained generator (its status must be **Ready**) and click **Generate synthetic data**.
        <Image 
            src="/docs/docimages/synthetic-datasets/create/01-from-generator-click-generate.webp" 
            alt="MOSTLY AI - Synthetic datasets - Create a new synthetic dataset from inside a generator" 
            width={800} 
            height={150}
        />
    * From the **Synthetic datasets** page, click **+ New synthetic dataset** and select a generator.
2. (Optional) On the **Synthetic dataset configuration**, configure how you want to generate.
    For details on specific configurations, see the related pages.
        * [Set sample size and temperature](set-size-temp)
        * [Data imputation](data-imputation)
        * [Rebalance columns](rebalance-columns)
        * [Deliver to databases or cloud buckets](deliver)
3. Click **Start generation**.
    <Image 
            src="/docs/docimages/synthetic-datasets/create/02-click-start-generation.webp" 
            alt="MOSTLY AI - Synthetic datasets - Click Start generation" 
            width={800} 
            height={150}
        />
</Tabs.Tab>
<Tabs.Tab>
When using the [Python SDK in a local environment](/python-sdk#use-in-a-local-environment), use `mostly.generate` to create a synthetic dataset.

To generate only a certain number of synthetic samples, use the `size` parameter.

```python copy filename="python"
sd = mostly.generate("GENERATOR_ID", size=1_000)
sd.data()
```

To generate the same size as the original data, drop the `size` parameter. The default sample size of the original dataset will be applied.
```python copy filename="python"
sd = mostly.generate("GENERATOR_ID")
sd.data()
```
</Tabs.Tab>
<Tabs.Tab>
With the [Python SDK connected to a remote platform](/python-sdk#use-with-a-remote-mostly-ai-platform), again use `mostly.generate` to create a synthetic dataset.

To generate only a certain number of synthetic samples, use the `size` parameter.

```python copy filename="python"
sd = mostly_remote.generate("GENERATOR_ID", size=1_000)
sd.data()
```

To generate the same size as the original data, drop the `size` parameter. The default sample size of the original dataset will be applied.

```python copy filename="python"
sd = mostly_remote.generate("GENERATOR_ID")
sd.data()
```
</Tabs.Tab>
